 # Calendar App

Proyecto fullstack MERN realizado como práctica siguiendo un curso de React de Fernando Herrera en Udemy, enfocado en simular un calendario y aprender el flujo completo de desarrollo web.

## Development

1. Instalar dependencias `pnpm i`
2. Cambiar variables de entorno acorde a tu config
3. Correr servidor `pnpm start`


## 🛠️ Tecnologías utilizadas

* React 19
* Vite
* Node
* bcrypt
* JWT
* express
* moment
* mongoose
* MomentJS
* nodemon
  

## 🧠 Conceptos y funcionalidades trabajadas

* Persistencia de sesión y autenticación con JWT
* Encriptación de sesiones
* Modelos y controllers
* CRUD completo de eventos de calendario
* Middlewares de Express
* Utilización de MongoDb, con MongoCompass
* Integración frontend/backend mediante APIs REST
* Despliegue completo de frontend, backend y base de datos en la nube

## 🛠️ Funcionalidades implementadas

* Autenticación de usuarios
* Login persistente
* CRUD de eventos
* Modal de eventos
* Despliegue fullstack en producción


## ⚠️ Notas

Este proyecto tiene fines educativos y de práctica. Algunas funcionalidades pueden estar simplificadas y no reflejar un entorno de producción real.

El stack está desplegado en railway (plan gratuito), por lo que puede tardar unos segundos en iniciarse tras un periodo de inactividad.
